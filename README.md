# Useful Skills

This repository contains a collection of skills, tools, and resources for various tasks including:

- **Agent Capabilities**: Agent identifier, memory, and skill creation tools.
- **Design & UI/UX**: Canvas design, UI design systems, frontend guidelines, and UX research.
- **Marketing & Content**: Copywriting, conversion optimization, marketing strategy, and storytelling.
- **Product Management**: PM toolkits and team utilization guides.
- **Development**: Frontend design, n8n MCP tools, and component caching.
- **Research**: Web research, lead research, and competitive landscape analysis.

## Structure

Each directory represents a specific skill or domain area containing relevant documentation (`SKILL.md`), scripts, and resources.
